Microsoft comprou o GitHub por 7.5 bilhões de dólares. Acredita-se que a Microsoft tenha decido construir as tão desejadas e necessárias pontes entre as diferentes plataformas. Entende-se que a aquisição do GitHub foi a maneira da Microsoft por na prática o discurso de melhoria de relacionamentos com desenvolvedores fora do Windows.


É possível gerenciar os reposítorios, arquivos, commits, times e outros aspectos pela plataforma Web. O site do GitHub é repleto de funcionalidades e oferece uma alternativa para o manuseio das ativadades relacionadas à ele.
