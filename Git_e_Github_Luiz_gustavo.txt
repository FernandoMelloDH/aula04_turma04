1 - Quem inventou o sistema de controle de versão Git e por quê?
O Git foi criado por Linus Torvalds, que é mais conhecido por outra criação sua, o Linux.
Ele densevolveu pela insatisfação de uso do BitKeeper, outro sistema de controle de versão que era utilizado pelos desenvolvedores do Linux.

2 - Quem atualmente é o dono do Github e por quê?
A Microsoft, após comprar por 7,5 bilhões de dólares. Ela comprou por conta de que o github precisava vender ou abrir capital e então seria interessante para a Microsoft ter mais uma empresa baseada em Open-Source.

3 - Existe outra maneira, além do terminal, de trabalhar com o Github?
Sim, existe Git GUI para quem quiser uma interface visual, algo mais amigável.
