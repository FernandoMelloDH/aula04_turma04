1. Quem inventou o sistema de controle de versão Git e por quê?
O inventor do Git foi Linus Torvalds, que é mais conhecido por ter criado o Linux.
Após uma mudança de licença de utilização do BitKeeper (até então o sistema de controle de versão usado por sua equipe para desenvolvimento do Linux), Torvalds desenvolveu o Git.


2. Quem atualmente é o dono do Github e por quê?
Desde 2018 a dona do Github é a Microsoft, após comprar a plataforma por US$7,5 Bi.


3.Existe outra maneira, além do terminal, de trabalhar com o Github?
Existe uma versão gráfica do Github, o Github Desktop.
